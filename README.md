## ServertBoard
JSP&Servlet을 이용해서 게시판을 만들어본다.

- [X] 게시글 리스트를 볼 수 있다. (게시글 번호, 제목, 작성일)  
- [X] 게시글을 작성 할 수 있다.  
- [X] 게시글 내용을 확인 할 수 있다. (게시글 제목, 내용, 작성일)  
- [ ] 게시글 내용을 확인하는 페이지에서 수정 및 삭제를 할 수 있다.  
- [ ] 게시글 수정을 누르면 해당 내용에대한 데이터가 바인딩 되어 페이지에 보인다.  
- [X] 게시글을 삭제 할 수 있다.
